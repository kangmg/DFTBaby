DFTBaby Documentation
========================================

**Tight-binding DFT calculations for ground and excited states with non-adiabatic molecular dynamics**

.. note::
   This documentation is for DFTBaby v0.1.0, which has been migrated to **Python 3.12+** and **NumPy 2.0+**.
   For migration details, see `PYTHON3_MIGRATION.md <https://github.com/kangmg/DFTBaby/blob/main/PYTHON3_MIGRATION.md>`_.
